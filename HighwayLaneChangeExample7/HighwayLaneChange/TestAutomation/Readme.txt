This directory contains the following files:

1. HighwayLaneChangeMetricAssessments.mldatx 	[Simulink test file]
2. HighwayLaneChangeMetricAssessments.slmx	    [Traceability data file]
3. HighwayLaneChangeTestRequirements.slreqx 	[Requirements data file]

These files are related to automate testing of Highway Lane Change test bench model. 
If you have the licenses to Simulink Requirements and Simulink Test, then this files can be used to run the test scenarios in automated way.

For more details, open the example using:
openExample('autonomous_control/AutomateTestingForHighwayLaneChangeExample');
